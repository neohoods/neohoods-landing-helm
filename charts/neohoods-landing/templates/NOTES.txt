1. Get the application URL:
   https://{{ .Values.domain }}

2. Monitor the deployment:
   kubectl get pods -l app={{ .Release.Name }}-frontend -n {{ .Release.Namespace }}

3. View logs:
   kubectl logs -l app={{ .Release.Name }}-frontend -n {{ .Release.Namespace }}

4. Port forward for local testing:
   kubectl port-forward -n {{ .Release.Namespace }} svc/{{ .Release.Name }}-frontend 8080:80
